# Генерация иконок для PWA

## Быстрый способ

1. Откройте `generate-icons.html` в браузере
2. Иконки сгенерируются автоматически
3. Нажмите кнопку "Скачать" под каждой иконкой
4. Сохраните файлы с правильными именами в этой папке

## Необходимые размеры

- icon-72.png (72x72)
- icon-96.png (96x96)
- icon-128.png (128x128)
- icon-144.png (144x144)
- icon-152.png (152x152)
- icon-192.png (192x192)
- icon-384.png (384x384)
- icon-512.png (512x512)

## Альтернатива: Использовать свой дизайн

Вы можете создать свой дизайн иконки и использовать онлайн-генератор:

1. Создайте квадратное изображение 512x512px
2. Используйте https://realfavicongenerator.net/ или https://www.pwabuilder.com/imageGenerator
3. Загрузите сгенерированные иконки в эту папку
